# HAI-API

企业AI模型网关与令牌管理系统

## 项目简介

HAI-API 是一款专为企业内部AI中台设计的统一模型网关与令牌管理系统。

## 技术栈

- Java 17 + Spring Boot 3.2.5
- PostgreSQL + Redis
- JWT 认证

## 快速开始

```bash
# 编译
mvn clean compile

# 运行
mvn spring-boot:run -Dspring-boot.run.profiles=dev
```
